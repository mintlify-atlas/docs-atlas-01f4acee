---
title: "Quickstart: Get started with Relationship Visualizer"
description: "Download the Relationship Visualizer, open the workbook, enter data in the data worksheet, and render your first Graphviz diagram in under five minutes."
---

The Relationship Visualizer is a macro-enabled Excel workbook (`Relationship Visualizer.xlsm`) distributed as a free ZIP download from SourceForge. Once you have Graphviz installed and the workbook open, you can produce your first diagram by filling in two columns of data. This guide walks you through the entire process from download to rendered graph.

<Note>
Installation steps differ between Windows and macOS. Follow the appropriate guide before opening the workbook: [Install on Windows](/installation/windows) or [Install on macOS](/installation/macos).
</Note>

## Steps

<Steps>
  <Step title="Download and extract the workbook">
    Download `RelationshipVisualizer.zip` from [SourceForge](https://sourceforge.net/projects/relationship-visualizer/files/latest/download) — it is free and virus-scanned.

    Extract the ZIP to a local folder. The archive contains:
    - `Relationship Visualizer.xlsm` — the main workbook
    - `ExcelToGraphviz.applescript` — required for macOS only
    - `samples/` — ready-to-open example workbooks
    - License files

    **Windows users:** Before opening the file, right-click `Relationship Visualizer.xlsm`, select **Properties**, and check **Unblock** at the bottom of the General tab if the checkbox appears. This removes the Windows security block applied to files downloaded from the internet. See the [Windows install guide](/installation/windows) for full details.
  </Step>

  <Step title="Open the workbook and enable macros">
    Double-click `Relationship Visualizer.xlsm` to open it in Excel.

    When Excel displays a security warning — `SECURITY WARNING Macros have been disabled` — click **Enable Content**. The Relationship Visualizer is built entirely on VBA macros; the workbook cannot function without them.

    <Warning>
    Only enable macros for files you trust. Download the Relationship Visualizer exclusively from the [official SourceForge page](https://sourceforge.net/projects/relationship-visualizer/) to ensure the file has not been tampered with.
    </Warning>

    Once macros are enabled, the **Graphviz** tab and **Launchpad** tab appear in the Excel ribbon.
  </Step>

  <Step title="Enter data in the data worksheet">
    Click the **data** tab at the bottom of the workbook to open the data worksheet. This is where you define the nodes and edges of your graph.

    Each row represents one relationship. At minimum, fill in two columns:

    | Column | Purpose |
    |---|---|
    | `Item` | The source node (e.g., `a`) |
    | `Related Item` | The target node — an edge is drawn from Item to Related Item (e.g., `b`) |

    For example, entering a single row with `Item = a` and `Related Item = b` produces this DOT code:

    ```dot
    digraph "Relationship Visualizer"
    {
        a -> b;
    }
    ```

    You can add more columns — **Label**, **Style Name**, **Tooltip** — to control how each node or edge is displayed. Start simple: two columns is all you need to render a working graph.
  </Step>

  <Step title="Enable the Automatic toggle and render">
    In the Excel ribbon, click the **Graphviz** tab. Find the **Automatic** toggle and make sure it is turned on.

    With **Automatic** enabled, the graph re-renders every time you edit a cell in the data worksheet. You will see the graph appear to the right of your data (or in the `graph` worksheet if you have configured a separate output sheet).

    To trigger a render manually at any time, click the **Refresh Graph** button in the **Graphviz** tab.

    <Tip>
    If the graph does not appear, check that Graphviz is installed and that `dot` is on your system PATH. Open a terminal and run `dot -V` — you should see a version number. If the command is not found, complete the installation steps for your platform first.
    </Tip>
  </Step>

  <Step title="Explore and customize">
    Once your first graph is rendering, try these next steps:

    - **Change the layout engine** — click `dot`, `neato`, `fdp`, or another engine button in the **Graphviz** tab to change how the graph is arranged.
    - **Add styles** — open the `styles` worksheet and define named styles, then reference them by name in the `Style Name` column of your data.
    - **View the DOT source** — click the `source` worksheet to see the exact DOT code generated from your data.
    - **Export the graph** — use the **Publish** button in the **Graphviz** tab to save the graph as SVG, PNG, PDF, or another format.
    - **Open a sample** — browse the `samples/` folder from the ZIP for ready-to-use workbooks covering org charts, network maps, timelines, and more.
  </Step>
</Steps>
