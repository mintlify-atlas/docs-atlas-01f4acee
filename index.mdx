The **Relationship Visualizer** is a free, cross-platform Excel workbook that transforms rows of data into professional Graphviz diagrams. Define nodes, edges, and styles directly in Excel cells, then render stunning org charts, dependency trees, network graphs, timelines, and more — all without writing a line of code.

## Key features

<AccordionGroup>
  <Accordion title="Multiple layout engines">
    Choose from eight Graphviz layout engines with a single click: **dot** (hierarchical), **neato** (spring model), **fdp** (force-directed), **sfdp** (large graph), **circo** (circular), **twopi** (radial), **osage** (array/cluster), and **patchwork** (squarified treemap).
  </Accordion>
  <Accordion title="Built-in Style Designer">
    The interactive Style Designer worksheet lets you compose Graphviz format strings visually — pick shapes, colors, gradients, fonts, border styles, and edge arrowheads — then save them as named styles for reuse across your data.
  </Accordion>
  <Accordion title="SQL data import (Windows)">
    Write SQL SELECT statements on the `sql` worksheet to import data from other Excel workbooks or Access databases. Chain multiple queries, use recursion for trees, and iterate over result sets — all without leaving Excel.
  </Accordion>
  <Accordion title="SVG animation & tooltips">
    Post-process exported SVG files using find/replace rules on the `svg` worksheet. Add hover tooltips, clickable node animations, and JavaScript-driven interactivity to any diagram.
  </Accordion>
  <Accordion title="Multilingual ribbon">
    The Ribbon interface ships in English (US/UK), French, German, Italian, and Polish. All translations are stored in editable worksheets so you can correct or extend them.
  </Accordion>
  <Accordion title="JSON data exchange">
    Export and import workbook data, styles, and settings as JSON for version control, sharing, or programmatic generation of diagram content.
  </Accordion>
</AccordionGroup>

## Platform support

<Note>
The Relationship Visualizer runs on both **Microsoft Windows** and **Apple macOS**. SQL queries and Clipboard features are available on Windows only. Core diagramming — entering data, applying styles, rendering graphs, and exporting files — works fully on both platforms.
</Note>
